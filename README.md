# Simple Health Checker for Linux

Утилита для мониторинга состояния системы. Написана на Python в рамках обучения DevOps (Неделя 1).

## Возможности
- Проверка свободного места на диске.
- Мониторинг статуса службы Nginx (через `systemctl`).
- Логирование результатов в системный файл `/var/log/health_checker.log`.

## Установка и настройка

Для настройки прав просто запустите ./setup.sh

### Подготовка системы
Убедитесь, что у вас установлен Nginx и настроен Firewall:
```bash
sudo apt update && sudo apt install nginx -y
sudo ufw allow 'Nginx Full'
sudo ufw enable
```

## Настройка прав доступа (Security)
В данном проекте реализовано разграничение прав согласно иерархии FHS:
* **Логи:** Файл хранится в `/var/log/`, что соответствует стандартам Linux.
* **Доступ:** Для безопасности мы не используем `root`. Права на файл лога переданы конкретному пользователю через `chown`, а права на исполнение скрипта ограничены `chmod 755`.
